# Site de casamento - Delano & Istephany

Abra `index.html` no navegador ou acesse o servidor local já iniciado:

`http://127.0.0.1:5173/`

## Dados configurados

- Monograma: `D&I`
- Nomes: `Delano & Istephany`
- Data: `26 de dezembro de 2026`
- Horário: `16h00`
- Programação: recepção às `16h00`, cerimônia às `16h30`, jantar e festa às `17h30`
- Local: `Sítio do Almir, bairro Pedra Branca, Pacajus-CE`
- Área de presentes: barra única para `Pix dos noivos`

## Recursos adicionados

- A barra de Pix é clicável e abre uma aba/modal editável no ADM.
- A aba de Pix tem campo para texto personalizado e upload de foto com prévia.
- Os textos das abas ficam salvos no navegador com `localStorage`.
- O cursor foi personalizado via CSS nativo para manter movimento fiel, sem atraso de JavaScript.
- As abas têm animação de abertura e fechamento com brilho; folhas aparecem apenas na entrada.
- A barra de rolagem foi estilizada na paleta verde-oliva, branco e cinza.
- O clique cria uma animação rápida de pétalas/brilho.
- A música `Viva La Vida` foi adicionada em loop, com fade-in de 3 segundos.
- O botão de música abre uma configuração pequena para ativar/desativar e ajustar volume.
- O painel `ADM` no canto superior direito ativa edição dos textos principais direto na página.
- A seção `Memórias` tem quadros descontraídos para upload de fotos pelo modo ADM.

Observação: navegadores podem bloquear autoplay com som antes do primeiro clique. O site tenta iniciar automaticamente e, se o navegador bloquear, tenta novamente assim que o visitante interagir.
